Paintings using GANs

This stye of art uses neural style transfer which is a pretty new concept. It involves making GANs learn the style to be transferred to a pre existing image. Well I thought, if that is the case let us try and change it to create a series of GANs that can keep syncing with each other to the point that the style is completely transferred after image corrections have been done. Maybe I cannot paint but I can sure make my AI learn how to paint like a famous painter!
