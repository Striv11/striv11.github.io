WWe knew mobiles can charge each other and which led me to think, EV are going to be as ubiquitous as mobiles in the future. What if EVs can charge each other? My university has less parking space, what if I develop a scheme for sharing the cars? We did simulations for the implementation of the same in the academic setting of the Birla Institute of Technology and Science, using seven distinct destination points and fast charging posts based on a battery to battery charging system (B to B) where series charging is used. The system uses real world data collected from the college campus and a model was developed for the same to demonstrate the experiment that I conducted. A 3 level charging system is used for the same with ac, dc and fast charging method mode.
